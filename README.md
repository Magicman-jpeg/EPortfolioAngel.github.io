README: OOP Midterm E-Portfolio
A lightweight, single-page professional E-Portfolio designed to showcase academic Java projects and reflections for Object-Oriented Programming.

🚀 Features
Minimalist Design: Built with a clean, soft aesthetic using Poppins typography.
Dynamic Modals: Interactive cards that open to display project details without leaving the page.
Image Lightbox: Built-in previewer to view code screenshots and program outputs in full size.
Categorized Content: Organized sections for Quizzes, Assignments, Seatwork, and Activities.
Fully Responsive: Optimized for both desktop and mobile viewing.
